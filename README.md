# 太平洋房屋站前店房地合一試算

這是可直接部署到 GitHub Pages 的版本。

## 檔案內容
- `index.html`：網站首頁

## GitHub Pages 部署方式
1. 到 GitHub 建立一個新的 repository
2. 將這個資料夾內的檔案上傳到 repository 根目錄
3. 進入 GitHub 該 repository 的 **Settings**
4. 點 **Pages**
5. 在 **Build and deployment** 中選擇：
   - **Source**：Deploy from a branch
   - **Branch**：`main`
   - **Folder**：`/ (root)`
6. 儲存後，等待幾分鐘
7. GitHub 會提供一個公開網址，即可直接在手機 Safari 開啟使用

## 備註
- 若要自訂網址，可在 GitHub Pages 設定後再綁定自有網域
- 更新網站時，只要重新上傳並覆蓋 `index.html` 即可
